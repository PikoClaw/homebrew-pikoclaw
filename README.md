# homebrew-pikoclaw

Homebrew tap for [PikoClaw](https://github.com/PikoClaw/pikoclaw).

## Install

```bash
brew tap PikoClaw/pikoclaw
brew install pikoclaw
```

Or in one line:

```bash
brew install PikoClaw/pikoclaw/pikoclaw
```

## Usage

```bash
export ANTHROPIC_API_KEY=sk-ant-...
pikoclaw
```
